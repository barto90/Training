‚ö° Deployment Flow

  1. Deploy Infrastructure: ARM template creates AKS, ACR, and Log Analytics
  2. Attach ACR: Grant AKS permission to pull images from registry
  3. Get Credentials: Configure kubectl to connect to cluster
  4. Build Container: Create Docker image with PowerShell service
  5. Push to ACR: Upload image to private registry
  6. Deploy to K8s: Apply deployment manifests with kubectl
  7. Expose Service: Create LoadBalancer service for external access
  8. Monitor: Use Container Insights and Log Analytics
